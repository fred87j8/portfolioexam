GRUNDLÆGGENDE

WEB

Gennem analyse af forskellige stilarter har jeg dannet en forståelse for komposition, farvelære, fonts, hierarki og hvornår det giver mening at bruge hvad. Planlægningen tog form med moodboard og styletile, og ved brug af HTML og CSS fik jeg pustet liv i mit brutalistiske site, som var opbygget efter det udleverede layoutdiagram. Forløbet har dannet det fundament indenfor webdesign, jeg stadig bygger videre på.

02.01.02 STILPITCH

Arbejde med stilarten Brutalist website design

Gennem arbejdet med stilarten lærte vi begreber til at karakteriserer en stil; specifikt fandt vi ud af, at brutalistiske websites overvejende stræber efter et carefree look, som er kaotisk, funktionelt og råt. Det kommer ofte til udtryk gennem asymmetri, høj farvekontrast mellem elementerne, et minimum af negativ space og masser af referencer til 90’erne. Skriften er gerne karakteristisk, f.eks. en black serif font med både et dekorativt og funktionelt formål. Disse principper fulgte jeg i udarbejdelsen af det responsive site for at opnå den genkendelige stil.

02.04.02 RESPONSIVE SITE_v2

Udvikling af responsivt site

I temaet om web blev vi introduceret til basic HTML og CSS syntax. Eksempelvis lærte vi om forskellige selectors i CSS: selectoren angiver hvilket element på sitet, vi ønsker at at give properties (style). Selctors er f.eks: id (#fifth_section), class (.container), * for alle elementer, ét element (p, body), flere elementer (p div) eller elementer inde i elementer (p, div).Derudover lærte vi brug af media queries til at gøre sitet responsivt, dvs. optimeret til forsk. enheder.
Jeg måtte sande, at selv en dansklærer er mere tilgivende når det gæder kommafejl end CSS. Så det gav pote til studiestartsprøven at være detaljeorienteret.
